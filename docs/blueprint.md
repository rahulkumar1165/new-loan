# **App Name**: LoanFlow

## Core Features:

- Mobile Number Authentication: Securely authenticate users using their mobile number to begin the loan application process.
- PAN Verification: Verify the user's PAN card details for accurate identification.
- Basic Information Input: Allow users to input basic personal information like name, date of birth, and address.
- Document Upload: Enable users to upload necessary documents such as gas bills, water bills, or bank statements.
- Bank Account Addition: Allow users to add their bank account details to facilitate transactions.
- Employment and Salary Information: Collect employment and salary details from users to assess their eligibility.
- Video Verification Tool: Incorporate a video verification tool that provides a phrase which the user then speaks on camera, to enhance security and user identification.

## Style Guidelines:

- Primary color: Deep indigo (#3F51B5) to convey trust and professionalism.
- Background color: Very light grey (#F0F2F5) to create a clean, modern interface.
- Accent color: Teal (#009688) for important actions and highlights.
- Body and headline font: 'Inter', a grotesque-style sans-serif with a modern look.
- Use a consistent set of minimalist icons throughout the app for easy navigation.
- Ensure all screens are optimized for a 390px width for a crisp mobile experience.
- Incorporate subtle transitions and loading animations to enhance the user experience.
